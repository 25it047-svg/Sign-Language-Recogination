## Project Scope

The objective of this project is to recognize sign language gestures using Machine Learning techniques. The system identifies hand gesture patterns based on input features and predicts the corresponding sign language symbol. This project demonstrates the application of Artificial Intelligence in improving communication accessibility for hearing-impaired individuals.

## Technologies Used

- Python
- Google Colab
- Pandas
- Scikit-Learn
- Decision Tree Classifier

## Project Description

Sign Language Recognition is an AI-based system that recognizes hand gestures and converts them into meaningful sign language symbols. In this project, gesture data is used to train a Decision Tree Classifier model. The model learns patterns from the dataset and predicts the corresponding sign based on user input.
